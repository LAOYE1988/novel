# AI 智能体系统

> 工作台内置 9 个 AI 智能体，通过 DeepSeek API 驱动。当 AI 需要了解智能体角色和职责时查阅。

---

## 智能体总览

| 类别 | 智能体 | 职责 |
|:----|:-------|:-----|
| 主控 | 网文主笔-总控 | 节奏监控、大纲匹配 |
| 基础设定 | 设定管家-书库 | 人设一致性、伏笔追踪 |
| 基础设定 | 书风认知-投喂专员 | 文风记忆、同风格仿写 |
| 剧情架构 | 爆款解构-剧情分析师 | 爽点分析、节奏优化 |
| 剧情架构 | 大纲架构师-章纲 | 细纲生成、高潮设计 |
| 正文生成 | 正文写手-去AI | 场景扩写（纯参考） |
| 灵感创意 | 灵感引擎-爽点 | 卡文救急、反转冲突 |
| 审核优化 | 金牌责编-润色 | 去AI味、质感提升 |
| 写作素材 | 智能选词词库助手 | 分类描写词汇 |

---

## 调用方式

通过 Web 工作台（http://localhost:8899）的前端页面选择智能体发起对话，后端通过 `deepseek_api.py` 调用 DeepSeek API。

完整配置见 [tools/小说工作台/agents/agents-config.json](file:///d:/trae/novel/tools/小说工作台/agents/agents-config.json)
