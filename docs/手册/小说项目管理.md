# 小说项目管理

> 小说项目的创建、结构规范和管理规则。当 AI 需要操作小说项目时查阅。

---

## 标准目录结构

```
小说名/
├── 大纲/         # 总纲.md、分卷.md、逐章细纲.md
├── 文稿/         # 第N章.txt
└── 设定集/       # 世界观.md、人物.json、伏笔.json、关系图.md、场景.md
```

## 项目操作

- 创建新小说：`tools/import_agents.py --create-novel <名称>`
- 切换小说：`tools/import_agents.py --switch-novel <名称>`
- 删除小说：`tools/import_agents.py --delete-novel <名称>`
