# 项目架构

> 项目完整目录结构和各模块职责说明。当 AI 需要了解项目全貌、定位文件路径时查阅。

---

## 完整目录树

```
novel/
├── .trae/                          # TRAE IDE 配置
│   ├── mcp.json                    # MCP 服务器连接（openviking）
│   ├── rules/
│   │   ├── project_rules.md        # 顶层规则（行为约束）
│   │   └── 脑图.md                  # 项目思维导图
│   └── skills/
│       └── 小说git操作/SKILL.md     # Git 操作 Skill
├── tools/                          # 核心工具层（业务逻辑）
│   ├── config.py                   # DeepSeek API 配置
│   ├── deepseek_api.py             # API 调用封装
│   ├── board_manager.py            # 创作看板管理
│   ├── workspace_manager.py        # 工作区/小说项目管理
│   ├── novels_config.json          # 小说列表配置
│   ├── workspace_config.json       # 工作区路径配置
│   ├── .env                        # API Key（已 gitignore）
│   ├── agents/
│   │   └── import_agents.py        # CLI 工具
│   ├── 小说工作台/
│   │   ├── server.py               # Web 服务器（端口 8899）
│   │   ├── 编辑器/                  # 前端页面
│   │   ├── agents/
│   │   │   └── agents-config.json  # 9 个 AI 智能体
│   │   ├── 大纲/ 设定/ 文风/        # 工作台模板
│   └── _archive/                   # 旧版文件归档
├── 奇幻大陆/                        # 当前激活的小说项目
├── 混在女帝身边的假太监/             # 主要小说项目（119 章）
├── 异类/                            # 小说项目（仅设定）
├── 素材库/                          # 共享写作素材
├── docs/
│   └── 手册/                        # 业务手册（按需懒加载）
├── .env.example                    # 环境变量模板
├── .gitignore                      # Git 忽略规则
└── README.md                       # 项目说明
```

---

## 各模块职责

| 模块 | 职责 |
|:----|:-----|
| `.trae/` | TRAE IDE 配置、顶层规则、Skill 定义 |
| `tools/` | 核心业务逻辑：API 封装、看板、工作区、Web 服务 |
| `docs/手册/` | 业务手册，按需加载，不全局注入 |
| `小说目录/` | 各小说的大纲、文稿、设定集 |
| `素材库/` | 跨项目的共享写作素材 |
