# 变更记录 — 使用说明

每次 AI 执行修改后，会按日期追加记录，格式如下：

---

## YYYY-MM-DD（星期几）

### HH:mm — 操作类型

**修改说明：** 这里写改了什么东西

**涉及文件：**
- `文件路径` — 做了什么

---

**示例：**

## 2026-06-25（星期四）

### 21:50 — 规则修复

**修改说明：** 修复了顶层规则中 PowerShell 启动命令不兼容的问题

**涉及文件：**
- `.trae/rules/project_rules.md` — 启动命令分两行、补充 MCP 信息、修复文档创建矛盾
